# 🔐 Passor - AES 加密/解密工具

Passor 是一個基於網頁的 AES 加密/解密工具，提供安全、直觀的資料保護服務。

## ✨ 功能特色

- 🔒 **AES 加密/解密** - 使用業界標準的 AES 加密演算法
- 🎨 **雙主題模式** - 支援淺色/深色主題切換
- 🔄 **雙操作模式** - 加密模式和解密模式一鍵切換
- 📱 **響應式設計** - 完美適配桌面和行動裝置
- 🔑 **密碼強度檢測** - 即時反饋密鑰強度
- 📋 **一鍵複製** - 結果快速複製到剪貼簿
- 👁️ **密碼顯示切換** - 可切換密鑰顯示/隱藏
- 🎪 **優雅動畫** - 流暢的使用者介面動畫效果

## 🚀 快速開始

### 線上使用

1. 下載專案檔案到本地
2. 開啟 `index.html` 即可使用
3. 無需伺服器，支援離線使用

### 本地部署

```bash
# 克隆專案
git clone https://github.com/pcion123/passor.git

# 進入專案目錄
cd passor

# 使用任何 HTTP 伺服器啟動（可選）
# 例如使用 Python
python -m http.server 8000

# 或使用 Node.js
npx http-server
```

## 📖 使用說明

### 加密模式
1. 選擇「🔒 加密模式」
2. 在「原始資料」區域輸入要加密的內容
3. 設定密鑰（建議至少 8 個字符）
4. 點擊「🔒 開始加密」
5. 從彈出視窗複製加密結果

### 解密模式
1. 選擇「🔓 解密模式」
2. 在「加密資料」區域輸入要解密的內容
3. 輸入對應的密鑰
4. 點擊「🔓 開始解密」
5. 從彈出視窗複製解密結果

### 主題切換
- 點擊右上角的 🌙/☀️ 按鈕切換主題
- 系統會自動記住您的主題偏好

## 🔧 技術架構

### 前端技術
- **HTML5** - 語義化標記
- **CSS3** - 現代樣式和動畫
- **JavaScript (ES6+)** - 核心功能邏輯
- **CryptoJS 4.1.1** - AES 加密實作

- 純前端實作，無需後端伺服器
- 支援離線使用
- 資料不會上傳到任何伺服器
- 本地儲存主題偏好設定

### 🛡️ 安全性增強建議
為了獲得最佳安全性，建議：
1. **本地化 CryptoJS**：下載 CryptoJS 到本地而非使用 CDN
2. **檔案完整性驗證**：使用 SRI (Subresource Integrity) 驗證
3. **HTTPS 部署**：如果線上部署，務必使用 HTTPS

## 🔒 安全性說明

### ✅ 安全保證
- **本地加密**：所有加密/解密操作完全在使用者瀏覽器本地執行
- **零資料傳輸**：不會將密鑰或敏感資料傳送到任何遠端伺服器
- **開源透明**：所有程式碼公開可檢視，無隱藏後門
- **標準演算法**：使用 CryptoJS 庫實作業界標準 AES 加密
- **離線可用**：下載後可完全斷網使用，保護隱私

### ⚠️ 安全建議
- **強密鑰**：建議使用至少 12 位包含大小寫字母、數字和特殊字符的密鑰
- **環境安全**：確保在安全的電腦環境中使用，避免惡意軟體
- **定期更新**：建議定期下載最新版本以獲得安全更新
- **敏感資料**：極度敏感的資料建議使用專業的離線加密工具

### 🔍 安全技術細節
- **加密演算法**：AES (Advanced Encryption Standard)
- **加密模式**：CBC 模式（CryptoJS 預設）
- **金鑰衍生**：PBKDF2 金鑰衍生函數
- **隨機因子**：每次加密使用不同的初始化向量 (IV)

### 🛡️ 風險聲明
- 本工具適用於一般資料保護需求
- 對於高度機密資料，建議使用經過國家級認證的加密工具
- 使用者需自行確保操作環境的安全性
- 請妥善保管密鑰，遺失密鑰將無法復原資料
